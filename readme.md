Project Overview
This project predicts house prices using a Linear Regression model. It uses housing data and numerical features to estimate house prices.
Objective
Predict house prices based on housing features using Machine Learning.
Dataset
California Housing Dataset (Scikit-Learn)
Features:
Median Income
House Age
Average Rooms
Average Bedrooms
Population
Average Occupancy
Latitude
Longitude
Target:
House Price
Technologies Used
Python
Pandas
NumPy
Scikit-Learn
Workflow
Load Dataset
Preprocess Data
Handle Missing Values
Normalize Features
Split Train/Test Data
Train Linear Regression Model
Evaluate Performance
Predict House Prices
